# Kube-Proxy简述

```
运行在每个节点上，监听 API Server 中服务对象的变化，再通过管理 IPtables 来实现网络的转发
Kube-Proxy 目前支持三种模式：

UserSpace
    k8s v1.2 后就已经淘汰

IPtables
    目前默认方式

IPVS
    需要安装ipvsadm、ipset 工具包和加载 ip_vs 内核模块

```
